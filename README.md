# GPT Usage Intelligence & Token Economics Optimization System

## Overview
This project builds an enterprise-level analytics system to monitor GPT usage, token consumption, and AI cost efficiency across business units.

It simulates real-world LLM FinOps (Financial Operations for AI) used in modern AI-driven enterprises.

---

## Key Features
- GPT usage tracking across departments
- Token consumption and cost analysis
- GPT efficiency scoring model
- Waste detection in AI workflows
- Reuse and duplication analysis
- Power BI-ready datasets

---

## Tech Stack
- Python (Pandas, NumPy, Scikit-learn)
- SQL (aggregation logic)
- Power BI (dashboard visualization)
- Data simulation for enterprise AI usage

---

## Key Metrics
- Total token consumption
- Cost per department
- GPT efficiency score
- Cost per prompt
- GPT reuse ratio

---

## Visual Analytics
The project includes:
- Top GPT cost distribution charts
- Department-wise AI spend analysis
- Cost vs usage correlation plots
- Cost efficiency histograms
- Prompt-level cost optimization insights

These visuals simulate real enterprise AI FinOps dashboards used in production environments.

---

## Business Impact
- Reduces AI operational costs
- Identifies inefficient GPT workflows
- Improves AI resource allocation
- Enables enterprise AI governance

---

## How to Run
1. Open Google Colab
2. Run notebook sequentially
3. Export CSV files
4. Load into Power BI

---

## Output Files
- gpt_usage_logs.csv
- gpt_usage_summary.csv
- department_cost_analysis.csv
- gpt_efficiency_scores.csv

---

## Outcome
Transforms raw GPT logs into actionable financial and operational insights for enterprise AI optimization.
